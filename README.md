# notebooklm-ia-na-educacao
Neste projeto, desenvolvi um caderno temático sobre Inteligência Artificial na Educação utilizando o NotebookLM como ferramenta de aprendizagem ativa. O repositório apresenta o contexto do estudo, a curadoria de fontes abertas, os testes de prompts, as dificuldades encontradas durante o processo e um miniguia final com resumos
